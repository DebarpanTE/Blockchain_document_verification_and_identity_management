# Blockchain Identity Management System

A blockchain-based identity management system implemented with Python and Flask.

## Features
- Secure identity management using blockchain principles.
- User dashboard for managing personal data.
- Flask-based web interface.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Configure your `.env` file.
4. Run the application: `python main.py`.
